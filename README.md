Writing nested logs from GAE Flex (custom runtime) using Node.js + Winston
===
A simple app written for GAE Flex Custom Runtime following the instuctions from the official [
nodejs-logging-winston](https://github.com/googleapis/nodejs-logging-winston#using-as-an-express-middleware) library.

The Dockerfile in this sample is using the [Node.js base image](https://cloud.google.com/appengine/docs/flexible/custom-runtimes/build#base) provided by the GCP.
